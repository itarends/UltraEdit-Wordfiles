# UltraEdit-Wordfiles
# (c) Copyright by Roland Arends in 2015

Dieses Repository enthält Wordfiles, die man für das Syntax-Highlighting im UltraEdit und UltraEditStudio benötigt.

acucobol.uew           - Syntax-Highlighting für ACU Cobol GT von Microfocus
acucobol_xtanken.uew   - Syntax-Highlighting für ACU Cobol GT von Microfocus im Projekt X-tanken Int.

Die uew-Datei muss lediglich in %AppData%/Roaming/IDMComp/UltraEdit/wordfiles bzw. %AppData%/Roaming/IDMComp/UltraEditStudio/wordfiles abgelegt werden um das Syntax-Highlighting zu aktivieren.

Danach muss der UltraEdit neu gestartet werden.
